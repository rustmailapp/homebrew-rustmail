# RustMail Homebrew Tap

Homebrew formulae for [RustMail](https://github.com/rustmailapp/rustmail) — a self-hosted SMTP mail catcher with web UI, REST API, and CI assertions.

## Install

```bash
brew install rustmailapp/rustmail/rustmail
```

## Usage

```bash
rustmail serve
```

Open [http://localhost:8025](http://localhost:8025) to view the web UI. Send emails to `localhost:1025` via SMTP.

For full documentation, visit [rustmail.app](https://rustmail.app).
